# Hey, I'm Ahmad

I build AI-native operating systems where you talk to AI instead of filling out forms.

## What I'm Building

**[agentic-os](https://github.com/ahmadelswify/agentic-os)** is a collection of markdown-based operating systems powered by [Claude Code](https://claude.com/claude-code).

| OS | What It Does |
|---|---|
| **[work-os](https://github.com/ahmadelswify/agentic-os/tree/main/work-os)** | Professional task management, goal tracking, daily planning |
| **[career-os](https://github.com/ahmadelswify/agentic-os/tree/main/career-os)** | Resume tailoring, STAR stories, interview prep, application tracking |
| **[personal-os](https://github.com/ahmadelswify/agentic-os/tree/main/personal-os)** | Life goals, relationships, health, projects |
| **[meeting-os](https://github.com/ahmadelswify/agentic-os/tree/main/meeting-os)** | Meeting notes, decision tracking, action items, follow-ups |

Each OS is self-contained with its own `CLAUDE.md`, specialized agents, and slash commands. Clone it, point Claude Code at it, start talking.

## How It Works

Brain dump naturally. AI organizes everything. You keep working.

No databases. No SaaS. No vendor lock-in. Just markdown files on your machine and an AI that manages the system through conversation.
